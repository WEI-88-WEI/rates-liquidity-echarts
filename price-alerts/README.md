# Price Alerts

Simple price alert service with webhook-based phone escalation.

## Alert Channel

This project uses fwalert for phone call notifications.

- Trigger URL: `https://fwalert.com/32b74fca-cf54-4e72-84d9-3840041e8cda`

## Goal

Monitor configured price conditions and notify the user when triggers fire.

## Planned Features

- Configurable price conditions
- Multiple symbols or markets
- Webhook-based phone alerting
- Future support for additional notification channels
